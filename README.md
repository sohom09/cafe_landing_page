# ☕ Meet & Eat Café - Responsive Landing Page

## 📌 Project Overview

**Meet & Eat Café** is a modern, fully responsive one-page café website built using **HTML5, CSS3, and JavaScript**. The project showcases a premium coffee shop experience with elegant UI design, interactive components, online reservation functionality, and a shopping cart system.

This website was developed as part of a web development project focusing on responsive design, semantic HTML, modern CSS styling, and front-end user interaction.

## 🚀 How to Run

1. Clone the repository

```bash

```

2. Open the project folder

```bash
cd meet-and-eat-cafe
```

3. Launch the website

Simply open:

```text

```

in your preferred browser.
